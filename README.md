# 🧭 NAVMATE

### Design & Implementation of an Augmented Reality-Based Indoor Navigation System using GPS-Compass Sensor Fusion

🏆 Best Project Recognition – IEEE International Conference

📄 Research Paper Presented at ICSCAN

🎓 Final Year B.Tech Project – MVIT, Puducherry

## 📖 Overview

NAVMATE is a Flutter-based Android application designed to solve indoor navigation challenges in large buildings such as colleges, hospitals, shopping malls, and office complexes.

The system combines GPS and compass sensor fusion to calculate real-time directional guidance and displays navigation instructions through an Augmented Reality (AR) arrow overlay on the smartphone camera feed. NAVMATE integrates Firebase Cloud Firestore for real-time data management and provides role-based access for administrators and users.

The application supports room search, multi-floor navigation, campus mapping, visit history logging, arrival detection, and offline navigation without requiring additional infrastructure such as BLE beacons, RFID systems, or Wi-Fi fingerprinting.
